Export posts from your wordpress blog as PDF.

== Description ==

Export posts from your wordpress blog as PDF selecting some categories or downloading all posts,

<em>This plugin is dedicated to Deise Daiana Gugeler Bazanella, my wonderful wife.</em>

== Installation ==

1. Unzip `blog-as-pdf.zip` inside the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' admin menu in WordPress
1. Export your blog as PDF in the WordPress admin via `Configurations -> Blog as PDF`
